## Installation

Install @envie/utilities with npm

```bash
npm install @envie/utilities \
  --registry https://npm.pkg.jetbrains.space/bruhabruh/p/envie/npm/
```

Install @envie/utilities with yarn

```bash
yarn add @envie/utilities \
  --registry https://npm.pkg.jetbrains.space/bruhabruh/p/envie/npm/
```
